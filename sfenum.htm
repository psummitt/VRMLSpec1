SFEnum

Purpose: SFEnum is a single-value field found in the AsciiText, MaterialBinding, NormalBinding, Separator, ShapeHints, Texture2, and WWWAnchor Nodes. This field contains an enumerated type value.
Explanation

As with SFBitmask, nodes that use this field class define mnemonic names for the values. These values are written mnemonic enumerated type names. Again, the names differ among uses of this field in the various nodes.
