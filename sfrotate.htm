SFRotation

Purpose: SFRotation is a single-value field found in the OrthographicCamera, PerspectiveCamera, Rotation, and Transform Nodes. This field contains an arbitrary rotation. The SFRotation field is written as four floating point values separated by whitespace. These four floating point values represent the axis of rotation followed by the amount of right-handed rotation about that axis, in radians.
Example

0 1 0 3.14159265

This example describes a 180 degree rotation about the Y axis.
