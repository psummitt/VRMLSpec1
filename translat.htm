Translation

Purpose: This node is the last of the five matrix or transform property Nodes. It defines a translation by a three-dimensional vector.
Syntax

FILE FORMAT/DEFAULTS

Translation {translation 0 0 0 # SFVec3f}

Example Use

The following code fragment is from Allnodes.wrl.

Translation {translation 0 0 0}
