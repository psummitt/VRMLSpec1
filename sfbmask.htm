SFBitMask

Purpose: SFBitMask is a single-value field found in the Cone, Cylinder, and FontStyle Nodes. This field contains a mask of bit flags. Nodes that use this field class define mnemonic names for the bit flags. SFBitMasks are written as one or more mnemonic enumerated type names.
Syntax

( flag1 | flag2 | ... )

Explanation

If there is only one flag used, the parentheses are optional. The mnemonic enumerated type names described above will differ among uses of this field in various node classes.
