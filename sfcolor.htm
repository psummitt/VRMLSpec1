SFColor

Purpose: SFColor is a single-value field found in the DirectionalLight and PointLight Nodes. This node contains a color.
Explanation

The SFColor field is written as a RGB triplet of floating point numbers. These floating point numbers are written in standard scientific notation. They range in value from 0.0 to 1.0.
