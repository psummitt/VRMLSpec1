SFBool

Purpose: SFBool is a single-value field found in the DirectionalLight, PointLight, and SpotLight Nodes. This field containing a single Boolean value.
Explanation

Remember that Boolean values are either true or false therefore it can be written either as 0 (representing FALSE), 1 (representing TRUE), TRUE, or FALSE.
